## Local Setup

### 1. Clone

```bash
git clone https://github.com/Safa-Anan08/portfolio
```

### 2. Install

```bash
npm install
```

### 3. Environment Variables

Create:

```env
.env.local
```

Add:

```env
MONGODB_URI=your_mongodb_uri
```

---

### 4. Start Development

```bash
npm run dev
```

Open:

```bash
http://localhost:3000/
```
